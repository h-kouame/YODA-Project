==========================================================================
								ReadMe
==========================================================================

This is the FPGA code.

It is divided into three parts:

1) detector
2) pmodad1
3) sniffer

==========================================================================
								detector
==========================================================================

Contains all the code related to the detector module.

==========================================================================
								pmodad1
==========================================================================

Contains all the code related to the adc module

==========================================================================
								sniffer
==========================================================================

Contains all the code related to the sniffer. It makes use of the detector
and sniffer modules as entities and wires them correctly. It also includes 
the testbench code.
